________SOFTWARE__________ 

This software is written in Python, uses PyMySQL and Tkinter.  

Pre-requisites

1 - Minimum version Python 3.0.
2 - Install Tkinter library in Python 
3 - Install the PyMySQL library.
4 - Install MySqlWorkbench : follow the insrtuctions shown in the video bellow
			      https://www.youtube.com/watch?v=7S_tz1z_5bA
	
	
________FEATURES_____________________

1 - Suitable for Windows.
2 - Simple and easy to use GUI.
3 - Shows every output in the console area.
